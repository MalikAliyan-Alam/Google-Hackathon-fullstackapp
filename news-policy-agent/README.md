# PolicyAgent AI - Autonomous News/Policy Agent System

An autonomous multi-agent system that converts news articles and policy documents into structured insights, decisions, and simulated actions.

## Agents
- **Agent 1** — Content Ingestion & Chunking (Antigravity)
- **Agent 2** — Insight Extraction (Claude API)
- **Agent 3** — Decision & Action Recommendation (Claude API)
- **Agent 4** — Action Executor / Simulator (Antigravity)

## Stack
- Backend: Node.js + Express
- AI: Claude API (Anthropic)
- Frontend: Mobile Web App
- Orchestration: Antigravity Mission Control
